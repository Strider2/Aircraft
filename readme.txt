Aircraft 1.0

phpVMS module to list the aircraft you have under different airlines in your VA.

To install this file please follow these instructions:
1)install the sql file using phpmyadmin
2)upload the files to the right folders.
3)edit the aircraft.tpl file to suit your needs, and to add as many tabs as you wish to have or need.
4)create a folder in the lib>images folder called aircraft within that create a folder called thumbs
5)create your images and thumbs and upload them to those folders. remember the names as you will need them later.
6)create a link in your va <a href="<?php echo url('/Aircraft');?>">Aircraft</a>
